# KKT.ActivatedList

## Описание

Метод возвращает список касс, активированных в указанном периоде и закреплённых за указанным офисом. Если код офиса не передан, будет возвращён список касс, закреплённых за офисом текущего сотрудника.

Если в указанном периоде касса была активирована и продлена, то результат будет содержать 2 или более (если продлена более 1 раза) записей касс с одним и тем же РНМ.

Длина периода не должна превышать 7 дней.

## Параметры

Параметры передаются в формате записи со следующими полями:

| Название    | Тип          | Описание                                      |
|-------------|--------------|-----------------------------------------------|
| StartDay    | Дата         | Левая граница периода                         |
| FinishDay   | Дата         | Правая граница периода                        |
| OfficeCode  | Строка       | Код офиса, может быть не передан              |

## Результат

Результат - набор записей с полями:

| Название        | Тип          | Описание                                           |
|-----------------|--------------|----------------------------------------------------|
| KktRegID        | Строка       | РНМ кассы                                          |
| ActivationDate  | Дата         | Дата активации или продления кассы                 |
| AccountID       | Число целое  | Номер аккаунта, в состав которого входит касса     |
| INN             | Строка       | ИНН организации, использующей кассу                |
| LicenseID       | Строка       | Идентификатор лицензии, с которой связана касса (число или UUID) |
| EndLicenseDate  | Дата         | Дата действия лицензии для кассы                   |
| FSEndDate       | Дата         | Дата окончания срока действия ФН                   |
| FSID            | Строка       | Заводской номер ФН                                 |

            },
            "d": {
                "StartDay": "2018-01-21",
                "FinishDay": "2018-01-27",
                "OfficeCode": "1350"
            }
