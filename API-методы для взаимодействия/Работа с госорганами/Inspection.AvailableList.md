# Inspection.AvailableList

## Описание

Метод позволяет получить список инспекций заданного типа в заданном регионе (или во всей России, если в параметре Region передано значение null), к которым можно подключить данного контрагента (те инспекции, направления сдачи в которые были уже добавлены для данного контрагента, не входят в этот список).

## Параметры

| Название      | Тип          | Описание                                                                 |
|---------------|--------------|--------------------------------------------------------------------------|
| ContractorID  | Число целое  | Идентификатор контрагента                                                |
| InspectionType| Число целое  | Тип инспекции, возможные значения:<br>1 – налоговая инспекция,<br>2 – пенсионный фонд,<br>3 – территориальный орган гос. статистики,<br>7 – Росприроднадзор,<br>9 – ФСРАР |
| Region        | Строка       | Двузначный код региона                                                   |

## Результат

Набор записей со следующими полями:

| Название                | Тип       | Описание                                                                 |
|-------------------------|-----------|--------------------------------------------------------------------------|
| Name                    | Строка    | Название инспекции                                                       |
| Code                    | Строка    | Код инспекции                                                            |
| ExtraCode               | Строка    | Дополнительный код инспекции                                             |
| RequiredAgreementWithPF | Логический| Необходимо соглашение с пенсионным фондом (имеет смысл только для инспекций – пенсионных фондов) |

## Пример вызова

```json
{
    "jsonrpc": "2.0",
    "method": "Inspection.AvailableList",
    "params": {
        "ContractorID": 1126971809,
        "InspectionType": 1,
        "Region": "76"
    },
    "protocol": 2,
    "id": 0
}
