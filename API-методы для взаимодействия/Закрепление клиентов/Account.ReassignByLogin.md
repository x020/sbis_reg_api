# Account.ReassignByLogin

## Описание

Метод позволяет закрепить аккаунт и его контрагентов за указанным сотрудником. Если данный аккаунт является платным, то перезакрепляются все его контрагенты, если же закрепляемый аккаунт платным не является, перезакрепляются только те его контрагенты, которые не входят в платные аккаунты.

## Параметры

| Название  | Тип          | Описание                                                                 |
|-----------|--------------|--------------------------------------------------------------------------|
| AccountID | Число целое  | Идентификатор аккаунта                                                   |
| UserLogin | Строка       | Логин сотрудника, за которым нужно закрепить                             |
| Reason    | Строка       | Причина перезакрепления<br>Может быть передана пустая строка или null    |

## Результат

true, если действие выполнено успешно, иначе ошибка.

## Пример вызова

```json
{
    "jsonrpc": "2.0",
    "method": "Account.ReassignByLogin",
    "params": {
        "AccountID": 730157,
        "UserLogin": "teterin.sa",
        "Reason": "Увольнение сотрудника"
    },
    "protocol": 2,
    "id": 0
}
